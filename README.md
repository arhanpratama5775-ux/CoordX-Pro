# CoordX Pro 📍

**Location tracker for WorldGuessr & GeoGuessr** — Auto-detect Street View coordinates with detailed address info.

## 🎮 Supported Games

| Game | Status |
|------|--------|
| **WorldGuessr** | ✅ Working |
| **GeoGuessr** | ✅ Working (NEW!) |
| OpenGuessr | 🚧 Coming Soon |

## ✨ Features

- 🎯 **Auto-detect coordinates** from Street View
- 🗺️ **Interactive map** with location marker
- 📍 **Reverse geocoding** — get full address details
- 🔄 **Auto new round detection** — works in multiplayer!
- 📋 **Copy coordinates** with one click
- 🌙 **Dark theme** UI

## 📦 Installation

1. Download: https://github.com/arhanpratama5775-ux/CoordX-Pro/archive/refs/heads/main.zip
2. Extract the ZIP file
3. Open Chrome → `chrome://extensions/`
4. Enable **Developer mode** (top right)
5. Click **Load unpacked**
6. Select the extracted folder
7. Click extension icon to open side panel!

## 🚀 How to Use

1. Go to WorldGuessr or GeoGuessr
2. Start a game
3. Click the extension icon (side panel opens automatically)
4. Coordinates will auto-detect when Street View loads
5. Click **New Round** to reset for next round
